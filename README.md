# Spotify Database Design

This project contains a relational database ERD for a Spotify-style music streaming platform.

## Tools Used
- draw.io / diagrams.net

## Features
- Normalized relational database design
- Primary keys and foreign keys
- Many-to-many relationships using junction tables
- Tables for subscribers, subscriptions, playlists, libraries, songs, albums, artists, genres, and addresses
